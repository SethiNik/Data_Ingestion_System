About This Platform

This platform provides a scalable ingestion pipeline capable of extracting structured data from web tables and streaming the data into a relational database using Kafka.

The system automatically detects schema, streams data through Kafka, and dynamically stores results into MySQL tables.

Core Capabilities

  • Automatic HTML table extraction
  • Schema inference for unknown data
  • Kafka based streaming ingestion
  • Dynamic database table creation
  • End to end ingestion dashboard
  • Containerized deployment

Technology Stack

  • GoLang backend services
  • Apache Kafka streaming
  • MySQL storage layer
  • Docker container orchestration
  • REST API ingestion services
  • HTML CSS dashboard frontend

Design Goals

  • Reusable ingestion pipelines
  • Loose coupling via Kafka
  • Extensible schema support
  • Developer friendly deployment
  • Production oriented architecture
